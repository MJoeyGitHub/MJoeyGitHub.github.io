iOS访问系统通讯录

发表于 2016-05-04

苹果在iOS9以前有两个框架可以访问通讯录。总结一下瞅瞅

iOS9之后出了两个框架（方便快捷）

iOS9之后废弃了，上面这两个但是还可以用,只不过今后不会再维护罢了， 但是市面上的大部分的App应用还没有屌到最低配置iOS9的 ,所以有点尴尬，可以直接应用iOS9之前的，当然也可以根据不同的系统做两手抓，今后彻底废弃的时候不至于重新写！

Contacts、ContactsUI

Contacts是纯代码获取通讯录数据专用

使用场景为，自定义一些通讯录页面，可以操作通讯录里面的数据，展示不同的UI， 最常使用的UI展示页面为，获取通讯录的所有人的信息， 然后在UITableViewcell的右边加入按钮进行点击邀请其他人的操作

ContactsUI是直接调用系统的通讯录进行数据操作

使用场景为类似支付宝里面的充话费功能， 点击调出系统的通讯录， 然后点击通讯录的联系人将其电话号展示在自己的App内部

Contacts纯代码

首先在Info.plist配置
由于苹果安全策略更新，在使用 Xcode8 开发时，需要在 Info.plist 配置请求通讯录的相关描述字段（Privacy - Contacts Usage Description）然后在后面填写你要提醒的数字 例如：是否要您的App访问您的通讯录？

如果不配置Info.plist 会报如下错误，并且崩溃

This app has crashed because it attempted to access privacy-sensitive data without a usage description. The app’s Info.plist must contain an NSContactsUsageDescription key with a string value explaining to the user how the app uses this data.
