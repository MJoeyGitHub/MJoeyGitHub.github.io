Hexo的基本设置

发表于 2017-04-01

一直也没有一个像样的博客, 偶然机会得知还有此种神奇的博客，遂研究了一番，主要是根据一下两个博文搭建的博客，so 在此记录一下

这个是完美的配置教程

这个是如何去写博客的基本语法

安装Hexo

  • 安装node.js
  • 安装配置hexo

终端指令下载Hexo

  • npm install -g hexo-cli

如果出现问题 重新再前面加上 sudo npm install -g hexo-cli

  • hexo init blog （你的博客名字）
  • cd blog （你的博客名字）
  • npm install
  • hexo server
  • 以上操作会弹出一个网址http://localhost:4000/
  • 复制这个网址 到你的网页 发现你的博客搭建好了

创建你的博客

  • hexo new “创建我的题材”
  • hexo server 上传的hexo
  • 再进去 http://localhost:4000/ 看看是否有你创建标题的名字（右上方位）

找到你创建博客的文件夹 找到这个文件 用Xcode打开这个 vim _config.yml

关联github仓库URL

  • hexo generate (hexo g)
  • hexo deploy (hexo d)

然后再浏览器中输入http://你的账户名字.github.io/就行了

关联SSH key（此步骤是上传到GitHub， 如果只想在本地的话，放弃此步骤）

  • git config –global user.name “你的名字”
  • git config –global user.email “你的邮箱”

查看配置命令：

  • git config –list

生成SSH密钥：

  • ssh-keygen -t rsa -C “voidking@qq.com”

找到id_rsa和id_rsa.pub文件夹 ，打开id_rsa.pub，复制全文。打开https://github.com/settings/ssh ，Add SSH key，粘贴进去

再次部署

  • hexo d，根据提示输入用户名和密码，等待一会儿便成功了
