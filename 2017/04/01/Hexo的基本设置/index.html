#我的Hexo博客

这个是完美的配置教程

这个是如何去写博客的基本语法

3. 高亮一段代码[^code]

安装Hexo

    1. 安装node.js
    1. 安装配置hexo
      下载安装hexo:
      npm install -g hexo-cli

如果出现问题 重新再前面加上 sudo npm install -g hexo-cli
hexo init blog （你的博客名字）
cd blog （你的博客名字）
npm install
hexo server

以上操作会弹出一个网址 –> http://localhost:4000/
复制这个网址 到你的网页 发现你的博客搭建好了

  1. 创建你的博客
    hexo new “创建我的题材”
    hexo server 上传的hexo
    再进去 http://localhost:4000/ 看看是否有你创建标题的名字（右上方位）

4.配置Github

找到你创建博客的文件夹 找到这个文件 用Xcode打开这个 vim _config.yml
翻到最下面，改成我这样子的

deploy:
type: git
repo: https://github.com/你的账户名字/你的账户名字.github.io.git（助注意创建名字的时候 一定要用正好名字创建）
branch: master
其中上面的域名 是你在github 上面创新仓库的时自己创建的

“切记分号右边都要加空格”

  1. 关联github仓库URL
    hexo generate (hexo g)
    hexo deploy (hexo d)

  2. 然后再浏览器中输入http://你的账户名字.github.io/就行了
