cocococoapods结合Xcode的两种使用方式

发表于 2017-04-26 | 分类于 CocoaPods

本文介绍cocoapods与Xcode结合使用的两种使用方式

第一种

创建一个Podfile文件，输入你想要的第三方的代码

格式如下

右键你的工程文件 选择第二个选项 在终端上面输入

pod update –no-repo-update

第二种（推荐）

在终端上

cd 你的项目存储绝对路径

pod init

在你的项目文件出现的Podfile文件 ，填写内容

pod install

git指令上传项目到远程仓库
